Pre-request and System Environment

We provide a script to install all pre-request and setup all enviroment. Just download and run the script.

PPA

Add a file in /etc/apt/sources.list.d with follow content

  • For Ubuntu 12.04 Precise amd64
    deb http://s3-ap-southeast-1.amazonaws.com/thor.repository/ubuntu/dists/precise/main/binary-amd64 thor contrib
  • For Ubuntu 12.04 Precise i386
    deb http://s3-ap-southeast-1.amazonaws.com/thor.repository/ubuntu/dists/precise/main/binary-i386 thor contrib
  • For Ubuntu 13.10 Saucy amd64
    deb http://s3-ap-southeast-1.amazonaws.com/thor.repository/ubuntu/dists/saucy/main/binary-amd64 thor contrib
  • For Ubuntu 13.10 Saucy i386
    deb http://s3-ap-southeast-1.amazonaws.com/thor.repository/ubuntu/dists/saucy/main/binary-i386 thor contrib
